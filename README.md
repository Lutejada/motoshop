# motoshop API

1. Clonar el Proyecto
2. inicializar dependencias
   
   ```
   npm instal
   ```
3. clonar archivo ```.env.template``` y renombrar a ```.env```
4. Cambiar las Variables de entorno.
5. Levantar la base de datos.
    ```
    docker-compose up -d
    ```
6. Levantar el proyecto 
    ```
    npm run start:dev
    ```
